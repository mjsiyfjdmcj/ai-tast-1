# Brighters AI Chatbot

This project is a web-based chatbot for Brighters AI, a non-profit organization focused on climate issues. The chatbot is designed to be an intelligent and engaging assistant that can provide information about climate change, sustainability, and Brighters AI itself.

## Features

- **Bilingual Support:** The chatbot can communicate in both **Bangla** and **English**.
- **Language Switcher:** Users can easily toggle between languages using the buttons in the chat interface.
- **Default Language:** The chat starts in Bangla to cater to the primary audience.
- **Brighters AI Knowledge Base:** The chatbot is equipped with specific information about Brighters AI, including:
    - It is a non-profit organization.
    - It works on climate-related issues.
    - The founder is Sayedu Rahman Siam.
    - The chair is Faria Sultana Ami.
- **Fun & Engaging Personality:** The chatbot's responses are designed to be fun, friendly, and encouraging, creating a positive user experience.
- **Responsive Design:** The chat interface is built to work on various screen sizes.

## How to Run

No special setup is required. Simply open the `index.html` file in any modern web browser to start interacting with the Brighters AI chatbot.
